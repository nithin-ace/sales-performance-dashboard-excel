# 📊 Sales Performance Dashboard (Excel)

## Overview

This project analyzes retail sales data using Microsoft Excel to identify revenue trends, product performance, regional performance, and profitability insights.

The project demonstrates the complete analytics workflow from data validation and preparation to dashboard creation and business insight generation.

## Business Questions

This analysis was performed to answer the following questions:

* Which product category generates the highest revenue?
* Which region contributes the most sales?
* Which products perform best and worst?
* How do sales vary across different months?
* How does profit compare across regions and categories?

---

## Dataset Information

The dataset contains approximately 3,500 sales transaction records including:

* Order Date
* Product Name
* Category
* Region

## Data Preparation

The following steps were performed before analysis:

* Verified missing values and duplicate records
* Standardized date formats
* Extracted Month and Year fields for trend analysis
* Validated numerical and categorical data
* Calculated Profit Margin metrics
* Prepared data for Pivot Tables and dashboard reporting

## Key Findings

* Electronics generated the highest revenue among all categories.
* West region recorded the highest sales revenue.
* North region recorded the lowest sales performance.
* Camera was the highest revenue-generating product.
* May recorded the highest monthly sales revenue.
* Profit trends closely followed sales performance trends.

---

## Business Recommendations

* Continue investing in high-performing Electronics products.
* Analyze factors contributing to lower regional performance.
* Monitor seasonal sales patterns for inventory planning.
* Focus on improving performance of lower-selling products.
* Use regional insights to support targeted business strategies.
